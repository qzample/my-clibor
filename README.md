## Windows剪切板实现
### 技术栈： python + tkinter + sqlite3
### 实现功能点： 监听剪切板，剪切板持久化（sqlite3），队列数据结构的剪贴板（先进先出，队列大小为20），将剪切板内容添加到固定短语
